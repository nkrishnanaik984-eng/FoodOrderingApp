🍔 Online Food Delivery System

A beginner-friendly Java Full Stack Web Application that allows users to browse restaurants, explore menus, place food orders, and manage their accounts. This project was built to strengthen my understanding of Java web development concepts and follows the MVC architecture.

🚀 Features

- User Registration & Login
- Browse Restaurants
- View Restaurant Menus
- Add Items to Cart
- Update or Remove Cart Items
- Place Food Orders
- Order Confirmation
- Responsive and Simple User Interface
- Database Integration using MySQL

⚙️ Setup Instructions

1. Clone the repository.
2. Import the project into Eclipse.
3. Create a MySQL database.
4. Import the SQL file into MySQL.
5. Update the database credentials in the JDBC connection file.
6. Configure Apache Tomcat.
7. Run the project on the server.
